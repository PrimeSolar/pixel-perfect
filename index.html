Pixel Perfect Piece of Art

Piet Mondrian used CSS Grid before it was invented.
He's a Dutch painter who has created masterpieces
that you might have seen before.
And they look like the one that is called
Composition No. III with Red, Blue, Yellow and Black.
This piece was auctioned off for $50.6 million.
I've uploaded a graphic dimensions.jpg that breaks down the dimensions of each of the elements in the grid of the painting.
It's measured up with a ruler so that I can create a realistic pixel perfect version of this painting using CSS Grid and CSS Flexbox. You can see the result on this page.
